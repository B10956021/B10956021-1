資訊管理系

成立於民國82年，設二年制一班，83年增設研究所碩士班，於89年度起改招四年制，停招二技，90年增設研究所碩士在職專班，形成一個兼具研究與教學特色的新興科系。

教育目標

本系之學生培育目標定位為資管應用服務與整合系統，本系的 Slogan：建立積極的系 － 培育專業的學生（Professional Students From a Vigorous Department） 。旨在培養專業創新、理論與實務並重的資管技術人員和資管經理人才，即台灣資管領域中堅人才。藉由培養學生專業與創新能力之目標下，建立系所優質學習環境。具體而言，本系培養資管人才以市場需求為導向，配合政府產業發展之資訊科技應用需求及趨勢，因應南部港灣地區工商業之快速發展導向，在學術上注重基本理論與學習能力的培養；在實務上注重工作實務與產學交流，落實理論與學習能力的培養。
